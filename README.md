# WriteUp

WriteUp je aplikace, která umožňuje autorům publikovat jejich díla a dostávat zpětnout vazbu
od čtenářů formou komentářů pod jednotlivými kapitolami.

## Instalace

Pro lokální spuštění aplikace je nejprve třeba nainstalovat potřebné balíčky NPM.

```bash
PS C:<Cesta ke složce>\write-up-frontend> npm install
```

## Spuštění

Aplikace se poté spustí příkazem:

```bash
PS C:<Cesta ke složce>\write-up-frontend> npm run dev
```

Pokud je spuštění úspěšné, aplikace poběží na adrese http://localhost:5173/:

```bash
  VITE v7.2.2  ready in 3599 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```
